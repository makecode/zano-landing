Zano wallet

Please refer to this article to learn how install and create the wallet using Zano Desktop app. Zano is an open source project with code on available on Github. When you download Zano Wallet, you may want to verify its authenticity to make sure that you downloaded the correct one from the Hyle Team GitHub repository. More information can be found on resources page

Mining software

Mining software available for both CPU and GPU, with CUDA and OpenCL implementation. Source code is on Github. Apart from that Zano daemon features internal stratum-like server that can host miner clients via ethProxy protocol. How to solo mine guide is available on resources page
